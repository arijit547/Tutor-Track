# Push to GitHub

Your code is committed locally. To push to GitHub:

## Option 1: HTTPS (Recommended)
```bash
cd /home/arijit-paul/TutorAI
git push -u origin main
```
Enter your GitHub username and password/token when prompted.

## Option 2: SSH
```bash
cd /home/arijit-paul/TutorAI
git remote set-url origin git@github.com:arijit547/Tutor-Track.git
git push -u origin main
```

## Option 3: Personal Access Token
1. Go to: https://github.com/settings/tokens
2. Generate new token (classic)
3. Copy the token
4. Run:
```bash
cd /home/arijit-paul/TutorAI
git push -u origin main
```
Use token as password when prompted.
